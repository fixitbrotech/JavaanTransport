# Javaan Transport Website

I have developed a premium, high-performance website for **Javaan Transport** using **React, TypeScript, and Framer Motion**. The site follows a sophisticated **Orange and Black** color scheme with a focus on modern typography and smooth, scroll-triggered animations.

## 🏗️ Technical Stack
- **Framework**: React 19 + TypeScript (Vite)
- **Styling**: Vanilla CSS (Utility-driven methodology)
- **Animations**: Framer Motion (Optimized for scroll-based interactions)
- **Icons**: Lucide React
- **Routing**: React Router DOM

## 🎨 Design System
- **Colors**: Deep Midnight Black (`#050505`) and High-Visibility Blaze Orange (`#FF6B00`).
- **Typography**: `Outfit` for body (modern/clean) and `Bebas Neue` for headers (industrial/bold).
- **Aesthetics**: Glassmorphism effects, premium shadows, and interactive hover states (grayscale-to-color transitions).

## 📄 Page Overview

### 🏠 Home Page
- **Hero**: Atmospheric background with bold industry-leading typography and primary call-to-actions.
- **Service Snapshot**: Highlights key value propositions (Secure, Time-Critical, Nationwide).
- **Trust Elements**: Data-driven counter stats (15K+ Clients, 500+ Fleet) and premium star-rated testimonials.

### 📦 Services Page
- **Detailed Catalog**: In-depth explanations for Goods Transport, Logistics, Warehousing, and Local/Interstate delivery.
- **Alternating Layouts**: Keeps the user engaged with dynamic positioning of content and high-quality logistics imagery.

### 🚚 Fleet Page
- **Vehicle Showroom**: Showcases the capability of trailers, medium trucks, and mini pickups.
- **Spec Highlights**: Includes capacity (tons/KL), vehicle type, and insurance status.

### 📍 Tracking Page
- **Live Search**: Interactive tracking field with simulated database scanning animation.
- **Status Graph**: Visual roadmap of the shipment's journey from origin to destination.

### 💰 Get Quote Page
- **Personalized Calculator**: Comprehensive form capturing pickup/drop, load type, and weight.
- **Instant Estimate Engine**: Simulated backend logic that provides a price estimate with a 24h guarantee.

### 📞 Contact Page
- **Multichannel Support**: Direct links for Call, Email, and a stylized **WhatsApp** floating button.
- **Experts Form**: Specialized contact form for direct inquiries.
- **Maps**: Premium dark-themed map integration for the Mumbai headquarters.

### 🧾 About Us
- **Legacy Timeline**: A chronological journey of the company from 2010 to current industry leadership.
- **Core Values**: Showcases "Safety First", "Integrity", and "Innovation" through stylized badges.

## ✨ Key Features
- **Always-Enabled Animations**: Every section uses `Framer Motion` to animate into view as the user scrolls, both up and down.
- **SEO Optimized**: Fully configured `index.html` with meta tags, Open Graph properties, and descriptive titles.
- **Responsive Design**: Fluid layouts that transition seamlessly from desktop ultrawide to mobile screens.
